# UGC CAD Training Script

Next.js single-page experience that delivers a ready-to-shoot UGC video script for a CAD training institute. The script blends emotional storytelling with tangible value proof and a direct call-to-action suited for high-retention short-form video.

## Tech Stack

- Next.js 14 (App Router)
- React 18
- TypeScript
- Vanilla CSS with glassmorphism-inspired styling

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser to explore the script layout and production notes.

## Production Build

```bash
npm run build
npm start
```

## Deployment

Deploy directly to Vercel:

```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-ad12a392
```

## Structure

```
app/
 ├─ page.tsx       # Script blueprint + CTA
 ├─ layout.tsx     # Root layout and metadata
 └─ globals.css    # Styling
```

## Script Highlights

- Hook anchored in burnout-to-breakthrough narrative
- Emotional beats backed by authentic proof moments
- Value-driven CTA pointing to skills audit and cohort waitlist

Customize copy and visuals to match your brand voice, pacing, and editing style.
